# Sponsor Targets for Koala-Science (Scaled Edition)

Research output: 31 concrete sponsor targets for a future scaled-up edition of the koala-science peer-review-by-agents competition. Two categories: (1) API credits / model access for participants to run agents, (2) prizes for winners.

Ranked by ease of getting a yes — structured academic-credit programs first (Mila/McGill affiliation is a strong fit), then competitive grants, then ad-hoc DevRel/partnership asks.

## Tier 1 — Open academic credit programs

Apply directly via form. These are near-default yes for Mila-affiliated researchers.

## Skipped / dead ends

- **Cerebras Inference Research Grant** — closed Feb 17 2025, no reopening announced.
- **Replicate** — searched; no academic-credit program surfaced. Skip unless cold DevRel.
- **Sequoia / Accel** — couldn't find an analog of a16z's open-source grant program. Skip; they sponsor accelerators, not academic competitions.
- **Apple** — no public academic compute/hardware grant equivalent to NVIDIA's. Skip.
- **ICML / ICLR travel grants** — typically routed through the same financial-assistance channels as NeurIPS; same external-sponsor pattern (Jump, Citadel) applies.

## Recommended outreach order (concrete first 2 weeks)

1. **Apply Tier 1 in parallel** (rows 1–18) — these are forms, not negotiations. Aim for ~10 yeses.
2. **Email Mila partnerships** (#24) — leverage existing relationships.
3. **a16z** (#29) and **AI2** (#26) — cash + brand.
4. **Jump/Citadel travel grants** (#30, #31) — prize tier.
5. **Tier 2** (free tiers) — fall back on these without applications during the event.

## Caveats

- Most programs require a faculty PI; loop in a Mila/McGill faculty member early.
- xAI's academic program is closed-beta — verify Mila is in the partner list before relying on Grok credits.
- Cerebras's grant closed Feb 2025; check before relying on it.
- Apple, Replicate, Sequoia/Accel: no public academic-sponsor program found.
